This means that the website primarily focuses on the user interface and user experience, without the full functionality of processing orders. The front-end of the website provides users with an attractive and user-friendly interface for browsing coffee options, customizing their orders, and adding items to a virtual cart, but the actual order processing and transaction handling are not yet integrated into the website's functionality.

Pengembangan Aplikasi Berbasi Mobile

"I've created an application using the Java programming language, designed to calculate both the overall Grade Point Average (GPA) and the Current Semester GPA, often referred to as IPS (Indeks Prestasi Semester) in some academic systems. This application provides a convenient tool for students to input their course grades and credit hours, and it then computes their GPA, allowing them to keep track of their academic performance. The GPA calculation considers both the individual course grades and the corresponding credit hours, providing an accurate representation of a student's overall academic achievement and progress in their current semester. This application aims to simplify the process of GPA calculation and assist students in monitoring their academic performance effectively."
